# Definition: Invariant Points

- A point is invariant under a transformation if it is mapped to itself.

That is, if a point $( P(x, y) )$ is transformed to $( P'(x', y') )$, then:

\[
    P^′=P⇒(x^′,y^′)=(x,y)
\]

---

The Origin is Always Invariant (for Matrix Transformations)


## Identity
  - Invariant Points: All points
  - Matrix $\begin{bmatrix}
1&0\\
0&1  
\end{bmatrix}$

## Reflection in x-axis

  - Invariant Points: Entire x-axis
  - Matrix $\begin{bmatrix}
1&0\\
0&-1  
\end{bmatrix}$

## Reflection in y=x
  - Invariant Points: Line y = x
  - Matrix $\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}$
