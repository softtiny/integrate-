- Special Types of Matrices
    - Square matrix
    - Diagonal matrix
    - Identity Matrix
    - Upper Triangular matrix
        A square matrix said to be a Upper triangular matrix if $a_{i,j}=0$ if $ i > j$
    - Lower Triangular Matrix
        A square matrix said to be a Lower triangular matrix if $a_{i,j}=0$ if $ i < j$
    - Symmetric Matrix
    - Skew- Symmetric Matrix
